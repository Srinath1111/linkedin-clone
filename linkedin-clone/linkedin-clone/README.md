# LinkedIn Clone - Full Stack Assignment

This is a simple LinkedIn-like social app built with React (Vite) frontend, Node.js + Express backend, and MongoDB.

## Structure
- backend/
- frontend/

## Quick Start
1. Backend:
   - cd backend
   - cp .env.example .env
   - fill MONGO_URI and JWT_SECRET in .env
   - npm install
   - npm run dev

2. Frontend:
   - cd frontend
   - npm install
   - create .env with VITE_API_URL=http://localhost:5000/api (if needed)
   - npm run dev
